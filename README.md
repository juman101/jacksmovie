# Website - [MoviFlix](https://lucent-youtiao-42d69a.netlify.app/)


### How to Run Locally
Run the below command to clone the repo locally
```
git clone https://github.com/juman101/jacksmovie.git
``` 

### To run the application, run below commands 
```
npm install
npm run dev
```
